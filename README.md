# Лабораторна робота: Сайт-портфоліо студента ХАІ

Проєкт містить багатосторінковий персональний сайт-портфоліо, виконаний виключно на чистому **HTML5** без використання стилів CSS та скриптів JavaScript згідно з усіма вимогами семантичної розмітки.

## Структура проєкту:
* `index.html` — Головна сторінка (опис, цитата, зображення у тегу figure).
* `about.html` — Сторінка "Про мене" (біографія, складна таблиця навичок, вбудований відеофайєр).
* `contacts.html` — Сторінка "Контакти" (клікабельні лінки, інтерактивна форма зворотного зв'язку з валідацією полів).

## Реалізовані технічні вимоги:
1. Семантична структура (`<header>`, `<nav>`, `<main>`, `<section>`, `<aside>`, `<footer>`).
2. Обов'язкове використання тегів списків опису `<dl>` та таблиць `<table>`.
3. Валідація форм, унікальні `id`, атрибути `required` та `placeholder`.
4. Історія розробки містить понад 10 комітів за стандартом **Conventional Commits**.
