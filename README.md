# Лабораторна робота: Сайт-портфоліо студента ХАІ

Проєкт містить багатосторінковий персональний сайт-портфоліо, виконаний виключно на чистому **HTML5** без використання стилів CSS та скриптів JavaScript згідно з усіма вимогами семантичної розмітки.

## Структура проєкту:
* `index.html` — Головна сторінка (опис, цитата, зображення у тегу figure).
* `about.html` — Сторінка "Про мене" (біографія, складна таблиця навичок, вбудований відеофайєр).
* `contacts.html` — Сторінка "Контакти" (клікабельні лінки, інтерактивна форма зворотного зв'язку з валідацією полів).

## Реалізовані технічні вимоги (ЛР №1: HTML):
1. Семантична структура сторінок за допомогою тегів (`<header>`, `<nav>`, `<main>`, `<section>`, `<aside>`, `<footer_>`).
2. Обов'язкове використання списків опису `<dl>`, таблиць `<table>` та вбудованих медіа-елементів (`<video>`, `<figure>`).
3. Валідація форм, унікальні `id`, атрибути `required` та інтерактивні `placeholder`.

## Реалізовані технічні вимоги (ЛР №2: CSS):
1. Повна стилізація HTML-структури через підключення зовнішнього файлу `styles.css`.
2. Демонстрація 8 типів селекторів (елементи, класи, ідентифікатори, дочірні, атрибути, псевдокласи та псевдоелементи).
3. Налаштування блокової моделі (Box Model): відступи `margin`, `padding`, рамки `border` та скруглення кутів `border-radius`.
4. Реалізація фіксованої шапки сайту (`position: fixed`, `z-index`) та зв'язки позиціонування (`relative` + `absolute`) для статус-бейджа.
5. Підключення стороннього шрифту Roboto з Google Fonts та використання відносних одиниць виміру `rem`.
6. Оформлення інтерактивних елементів (ефекти `:hover`, `:active`, `:focus`) з дотриманням правила LVHA для посилань.
7. Керування вмістом блоків за допомогою властивості `overflow`.

## Історія розробки
Проєкт містить покрокові комміти за стандартом **Conventional Commits**, які відображають етапи створення структури сайту та його подальшої стилізації.
